# GUARDIAN - Missing Person Alert System

An AI-powered system that detects faces from images or CCTV footage and matches them with a database of missing persons.

## Features
- Face detection using OpenCV
- Face recognition using DeepFace
- Flask API backend
- SQLite database
- Twilio SMS alerts

## Tech Stack
- Python
- Flask
- OpenCV
- DeepFace
- TensorFlow
- SQLite
- Twilio
